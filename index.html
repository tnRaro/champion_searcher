왜 이런 걸 만들었나요?

조작법

Tab : 검색 토글

1, 2, 3 : 챔피언 선택, 두 번 연속으로 누를 시 챔피언 분석 보기

Q : 선택한 챔피언 분석 보기

W : 선택한 챔피언의 아이템 분석 보기

E : 선택한 챔피언의 스킬 분석 보기

R : 선택한 챔피언의 룬 분석 보기

Enter : 선택한 챔피언의 분석 보기 (검색 중에도 연결 가능)

콤보 예시

Note: 콤보는 기록된 데이터에 따라 달라질 수 있음에 유의

베인 + Enter : 베인 분석 보기

ㅂㅇ + 2 + R : 바이의 룬 분석 보기

wld + 3 + 3 : 징크스 분석 보기
